# 大学生数字阅读素养评估实验系统

## 项目概述

这是一个用于评估大学生数字阅读素养的完整实验系统。系统包含三个核心阅读任务，能够记录参与者的眼动数据（模拟）、交互行为和回答内容。

## 文件结构
digital-reading-assessment/
├── index.html # 实验入口页面
├── task1.html # 任务1：信息定位与筛选
├── task2.html # 任务2：多源信息整合
├── task3.html # 任务3：批判性评估
├── results.html # 实验结果页面
├── style.css # 主样式文件
├── script.js # 实验主逻辑
├── data-logger.js # 数据记录模块
├── aoi-manager.js # AOI（兴趣区域）管理模块
└── README.md # 使用说明

## 快速开始

1. 将所有文件放在同一个文件夹中
2. 使用现代浏览器打开 `index.html`
3. 按照页面提示完成实验
4. 数据会自动保存在浏览器的本地存储中

## 实验流程

### 1. 开始页面 (`index.html`)
- 参与者阅读并同意知情同意书
- 填写个人信息（参与者编号、年龄、性别、专业）
- 查看实验任务说明
- 开始实验

### 2. 任务1：信息定位与筛选 (`task1.html`)
- 阅读一篇关于社交媒体与青少年心理健康的学术文章
- 从侧边栏推荐文章中选择最相关的一篇
- 回答关于文章核心结论的问题
- 评估信息定位和筛选能力

### 3. 任务2：多源信息整合 (`task2.html`)
- 在两个标签页中阅读关于AI教育的不同观点文章
- 整合两篇文章的信息
- 撰写综合性的总结分析
- 评估信息整合和辩证思维能力

### 4. 任务3：批判性评估 (`task3.html`)
- 阅读一篇关于红酒健康益处的网络文章
- 识别文章中的可疑信息、广告标识和来源问题
- 从多个维度评估文章可信度
- 撰写详细的批判性评估
- 评估批判性思维能力

### 5. 结果页面 (`results.html`)
- 显示实验完成状态
- 展示各任务的数据统计
- 提供数据导出功能
- 收集参与者反馈

## 数据采集

### 自动记录的数据类型

1. **基本数据**
   - 参与者个人信息
   - 实验开始和结束时间
   - 各任务完成时间

2. **交互数据**
   - 鼠标移动轨迹（模拟眼动）
   - 点击事件（位置、目标元素）
   - 滚动行为
   - 键盘输入（在文本框中）

3. **AOI（兴趣区域）数据**
   - 对关键区域的进入/离开时间
   - 在AOI内的停留时间
   - 访问次数统计

4. **任务表现数据**
   - 回答内容
   - 选择结果
   - 评分结果
   - 完成度

5. **元数据**
   - 浏览器信息
   - 屏幕分辨率
   - 页面访问历史
   - 时间戳

### 数据存储位置

所有数据保存在浏览器的本地存储中：
- `participantData`: 参与者信息
- `taskData_[participantId]_[taskId]`: 各任务数据
- `experimentEvents`: 全局实验事件
- `experimentFeedback`: 参与者反馈

## 数据导出

实验结果页面提供多种数据导出格式：
- **JSON格式**: 完整结构化数据
- **CSV格式**: 表格化的事件数据
- **文本格式**: 人类可读的摘要
- **全部数据**: 所有本地存储数据的完整备份
